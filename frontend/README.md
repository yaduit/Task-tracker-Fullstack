# Task Tracker Frontend

Frontend for the Task Tracker application.

## Tech Stack

- React (Vite)
- TailwindCSS
- Axios
- React Router
- Context API
- Jest + React Testing Library

## Features

- User authentication (login/register)
- Protected routes
- Task CRUD UI
- Admin dashboard
- Responsive UI
- API integration
- Component tests

## Setup

Install dependencies:

npm install

Create `.env` file.

Example:

VITE_API_URL=http://localhost:5000

Run development server:

npm run dev

## Running Tests

npm test
